# iot_cloud

## 1.Deploy
1. Start Zookeeper Server
2. Start Kafka Server
3. Start MQ Server
4. Copy Web_* To Tomcat Server WebApps Directory
5. Start Tomcat Server
6. Start Service

The above is a must. Besides, you may need Nginx+KeepAlive, dubbo-admin, redis, mysql, HBase
